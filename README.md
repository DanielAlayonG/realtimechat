1) Clonar el repositorio
2) Ejecutar en la consola de preferencia las dependencias del proyecto: npm i
3) Ejecuta el proyecto en este tenemos dos opciones: npm start o npm run dev (si quieres que detecte cambios)
4) Validar el puerto y la url en la que se esta ejecutando para manipular el proyecto
5) la base de datos esta en la nube no requiere instalacion
